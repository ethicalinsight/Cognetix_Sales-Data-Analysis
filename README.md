# Cognetix_Sales-Data-Analysis
# Sales Data Analysis

## 📌 Objective
Analyze sales data to identify top-performing products, regions, and overall sales trends.

## 📊 Dataset
- Source: [Sample Sales Data](https://www.kaggle.com/datasets/kyanyoga/sample-sales-data)

## 🛠️ Tools & Libraries
- Python, Pandas, NumPy, Matplotlib, Seaborn 

## 🚀 Steps
1. Load and clean dataset (remove nulls, duplicates).
2. Compute summary statistics (sum, mean, max, min).
3. Category-wise and region-wise analysis.
4. Generate 5–7 professional visualizations.
5. Summarize insights.

## 📈 Deliverables
- Google Colab Notebook
- Visualizations (bar, pie, line charts)
- Insights summary
